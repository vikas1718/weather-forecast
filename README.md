# Weather App

A modern weather application with real-time data, built with Flask and JavaScript.

## Features
- Real-time weather data from OpenWeatherMap API
- 24-hour hourly weather forecast
- 5-day weather forecast
- Temperature conversion (°C/°F)
- Responsive design
- Search any city worldwide
- Dynamic sky backgrounds with weather animations
- Loading states and error handling
- Dark/Light theme toggle

## Tech Stack
- **Backend**: Python Flask
- **Frontend**: HTML5, CSS3, JavaScript
- **API**: OpenWeatherMap

## Setup
1. Get OpenWeatherMap API key
2. Set `OPENWEATHER_API_KEY` in `.env`
3. Run: `python app.py`
4. Open http://127.0.0.1:5000

## Usage
- Search for cities
- Toggle temperature units
- View comprehensive weather info
